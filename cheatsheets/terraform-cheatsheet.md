# Terraform Cheat Sheet

This cheat sheet summarizes how Terraform was used in this project.

## Purpose

Provision AWS infrastructure using Infrastructure as Code.

## Commands Covered

- terraform init
- terraform fmt
- terraform validate
- terraform plan
- terraform apply
- terraform state list
- terraform output

## Infrastructure Created

- VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group
- EC2 Instance
- S3 Remote Backend

## Outcome

Infrastructure was provisioned successfully and became ready for application deployment through Ansible.
